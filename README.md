## 🚀 Run Locally

No build step needed — just open in browser:

```bash
# Clone the repo
git clone https://github.com/jahnvithakur13/portfolio.git

# Navigate to folder
cd portfolio

# Open in browser
open index.html
# or just double-click index.html
```

For live reload while editing, use the **Live Server** extension in VS Code.

## 🌐 Deploy on Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **New Project**
3. Import your GitHub repo
4. Click **Deploy** — done in 30 seconds ✅
