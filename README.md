# 🏴‍☠️ Jahnvi Singh - Portfolio Website

> *"The sea is vast — so are the possibilities."*

A **cosmic One Piece-themed** personal portfolio built with pure HTML, CSS, and JavaScript. No frameworks, no build tools, just clean code.

🌐 **Live Site:** [jahnvithakur13.github.io/portfolio](https://jahnvithakur13.github.io/portfolio)

---

## ✨ Features

- 🌌 **Live cosmic star field** — animated canvas with twinkling stars, shooting stars & nebula clouds
- 🏴‍☠️ **One Piece aesthetic** — floating Straw Hat, Going Merry ship, Log Pose compass & Jolly Roger
- 🖱️ **Mouse parallax** — One Piece elements react to cursor movement
- 🌊 **Ocean wave dividers** — SVG wave transitions between sections
- 📱 **Fully responsive** — works on mobile, tablet & desktop
- ⚡ **Zero dependencies** — no frameworks, no npm, just HTML + CSS + JS
- 🎨 **Glassmorphism cards** — frosted glass UI with hover animations
- 🔤 **Space Grotesk typography** — bold modern font with gradient shimmer on name

## 🛠️ Tech Stack

* **HTML5** — Semantic structure and modern web markup
* **CSS3** — Responsive design, animations, glassmorphism effects, and custom layouts
* **JavaScript** — Interactive UI, canvas-based star field, parallax effects, and scroll animations
* **Font Awesome 6** — Professional icons for navigation and contact sections
* **Google Fonts** — Space Grotesk, Syne, DM Mono, and DM Sans typography
* **GitHub Pages** — Portfolio deployment and hosting
